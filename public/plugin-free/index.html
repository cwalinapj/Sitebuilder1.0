Install AI-WebAdmin Plugin

Step 1 of 4

Download the plugin ZIP, then install it in WordPress:

  1. In WordPress admin, go to Plugins -> Add New -> Upload Plugin.
  2. Upload ai-webadmin.zip.
  3. Click Install Now and then Activate.

Activation lock: Cloudflare + GitHub tokens must be verified and uploaded to Worker vault before features run.
