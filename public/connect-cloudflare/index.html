Connect Cloudflare

Step 2 of 4

  1. Sign up/sign in to your Cloudflare account and add your domain if not already added.
  2. Create a scoped API token for this site/account.
  3. In WordPress go to Settings -> AI WebAdmin -> General.
  4. Paste token into Cloudflare API Token (paste to verify/connect).
  5. Click Save Changes to send it to Worker for verification.
  6. Then go to Sandbox & Secrets tab and upload Cloudflare + GitHub tokens to Worker vault.

Plugin activation remains locked until both Cloudflare and GitHub vault tokens are set.
