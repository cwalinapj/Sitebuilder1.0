Install TollDNS

Step 3 of 4 (required for free tier)

  1. Download TollDNS plugin zip.
  2. In WordPress, go to Plugins -> Add New -> Upload Plugin, then upload and activate it.
  3. Open Settings -> TollDNS and set your monitored domain + nameservers.
  4. Run a nameserver check and confirm points are increasing.
  5. Return to AI-WebAdmin and complete Cloudflare connect verification.
